# Falling Sand
Pyxelを使用したピクセルシミュレーションゲームです。

## Pyxelのインストール

このプロジェクトは **Pyxel 2.5.11** を使用しています。

### pip を使用する場合
```
pip install pyxel==2.5.11
```

### Poetry を使用する場合
```
poetry install
```

## マテリアル一覧

| マテリアル | 説明 |
|-----------|------|
| Empty | 空のセル |
| Wall | 動かない壁 |
| Sand | 落下して積み重なる砂 |
| Water | 流動する液体 |
| Oil | 水より軽い油（水に浮く） |
| Fire | 燃焼して消える炎 |

## 実行方法
### ローカル環境での実行
pyxelインストール後、以下のコマンドで実行できます。
```
pyxel play .\app\main.pyxapp
```

## 操作方法

### メニュー画面
| キー | 操作 |
|------|------|
| ↑/↓キー | メニュー項目の選択 |
| Enter/Spaceキー | 決定 |

### ゲーム画面
| キー/マウス | 操作 |
|-------------|------|
| マウス左クリック | マテリアルを配置 |
| マウス右クリック | マテリアルを削除（空にする） |
| ↑/↓キー | ブラシサイズ変更（1〜10） |
| Spaceキー | ポーズ切り替え |
| Cキー | クリア |
| ESCキー | メニューに戻る |

### マテリアルセレクター
画面下部のマテリアルセレクターをクリックして、配置するマテリアルを選択できます。
