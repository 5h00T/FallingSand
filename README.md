# Falling Sand
Pyxelを使用したピクセルシミュレーションゲームです。


| マテリアル | 説明 |
|-----------|------|
| Empty | 空のセル |
| Wall | 動かない壁 |
| Sand | 落下して積み重なる砂 |
| Water | 流動する液体 |
| Oil | 水より軽い油（水に浮く） |
| Fire | 燃焼して消える炎 |

## 操作方法

### メニュー画面
- **↑/↓キー**: メニュー項目の選択
- **Enterキー**: 決定

### ゲーム画面
- **マウス左クリック**: マテリアルを配置
- **マテリアルセレクター**: 配置するマテリアルを選択
